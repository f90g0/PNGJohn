# PNGJohn
Twitterに圧縮されないPNGを出力するためのアプリケーション

## Overview
- Twitterに圧縮されないPNGを出力する
  - 3MBに収まるサイズまで画像を自動的に縮小する
  - 左上1pxにアルファチャンネル付きのピクセルを設定する
![image](https://i.imgur.com/R1FWbaU.png)
